# 可用工具和端点

## 代码仓库

- **路径**：`[YOUR_PROJECT: /path/to/repo]`
- **语言**：`[YOUR_PROJECT: TypeScript / Python / Go / ...]`
- **包管理器**：`[YOUR_PROJECT: pnpm / pip / go mod / ...]`

## 测试

- **运行命令**：`[YOUR_PROJECT: pnpm test --coverage]`
- **覆盖率输出**：`[YOUR_PROJECT: coverage/lcov-report/index.html]`
- **覆盖率阈值**：`[YOUR_PROJECT: 80%]`

## Lint

- **运行命令**：`[YOUR_PROJECT: pnpm lint]`
- **配置文件**：`[YOUR_PROJECT: .eslintrc.json]`

## 依赖安全

- **扫描命令**：`[YOUR_PROJECT: pnpm audit / pip-audit / govulncheck]`
- **忽略列表**：`[YOUR_PROJECT: .audit-ignore]`（如有）

## 输出位置

- **报告目录**：`runs/`
- **报告格式**：Markdown
- **文件名规则**：`YYYY-MM-DD.md`
